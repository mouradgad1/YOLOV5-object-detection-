# YOLOv5 Face Detection

A lightweight project demonstrating **YOLOv5** for detecting human faces using an open-source dataset from [Roboflow](https://roboflow.com/). This repository includes scripts for training, testing, and running inference on custom images.

## Features
- Real-time face detection with YOLOv5  
- Pretrained models for quick inference  
- Training on a curated open-source dataset  
- Easy-to-use scripts for testing images or videos

- Head to the Google Colab notebook and just run the cells
